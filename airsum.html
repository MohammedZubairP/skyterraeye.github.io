Room Program
Each room has a Position hint (Auto / Road-facing / Back / Side) — it steers which side of the plot the AI tries to place it on. Add a room and it regenerates immediately.
Manual editing: click any room, then drag its corner/edge handles to resize, or drag the body to move it — works immediately, no mode switch needed. Dragging a shared wall (e.g. Hall/Kitchen boundary) resizes the room on both sides together, no gaps or overlaps. DRAW ROOM adds a new rectangle by hand. SET SIZE types exact dimensions. Loads/columns/beams/footings recalculate automatically after every change.
