📐DWG File Import
✕
⚡
Import DWG Directly
Strata will parse the binary DWG file and extract geometry
✓ Supports R2000–R2025 (AC1015–AC1037)  ·  Extracts lines, polylines, arcs, circles, text
⚠ Civil 3D proxy objects and complex 3D solids may not extract fully
🔄 Or Convert to DXF First (Best Quality)
For Civil 3D files with surfaces, corridors, or proxy objects — convert to DXF first for 100% geometry fidelity.
📋 If you have AutoCAD / Civil 3D
For Civil 3D surfaces: AECCEXPORTTOACTUAD → converts proxy objects to LWPOLYLINE
Then: File → Save As → AutoCAD 2013 DXF (*.dxf) → drop into Strata
