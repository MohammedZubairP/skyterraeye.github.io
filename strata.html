🔒
Binary DWG cannot be opened directly
Strata reads ASCII DXF — the open exchange format. DWG is Autodesk's proprietary binary format and must be converted to DXF first. This takes <30 seconds with any of the free tools below.
Conversion Steps
1
Open your DWG in a converter
Choose any free tool from the list below — all of them can read your AC1027 file without needing a licence.
2
Export / Save As → DXF (ASCII)
Choose AutoCAD 2013 DXF or any ASCII DXF version. Avoid "Binary DXF" — it won't parse. In AutoCAD: File → Save As → AutoCAD DXF (*.dxf)
3
Import the .dxf back into Strata
Drag & drop the exported .dxf onto the canvas, or use File → Import Drawing. All layers, geometry, and blocks will be parsed automatically.
