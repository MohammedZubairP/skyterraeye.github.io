Capabilities

Everything in one pipeline

Powered by OpenDroneMap — the same engine used by professional mapping firms worldwide. No seat licence. No per-project fee.

Structure from Motion

FLANN feature matching + bundle adjustment across hundreds of overlapping images. Fast, medium, or accurate quality presets.

Dense Point Cloud

Multi-view stereo depth maps fused into a dense LAZ or PLY point cloud — ready for QGIS, CloudCompare, or AutoCAD.

DSM + DTM + Hillshade

Digital surface and bare-earth models at auto or custom resolution. SMRF ground filtering. GDAL hillshade overlay.

Georeferenced Orthomosaic

5 cm/px default resolution. EPSG projected, DEFLATE compressed, with radiometric balance and p2/p98 stretch.

Runs 100% Locally

Your images never leave the workstation. No API keys, no cloud subscription, no data transfer. Docker runs entirely offline after setup.

Works alongside Eye

Use Eye for thermal drone stitching and Terra for full RGB photogrammetry — both run simultaneously on the same machine, different ports.

One-time setup

Ready in 10 minutes

Terra uses Docker to run ODM — no Python environment needed beyond Flask. Install once, use forever.

1

Install Docker Desktop

Free download for Windows and Mac. Installs the Docker engine that runs ODM in an isolated container.

Docker Desktop
2

Download Terra agent

Download the agent files below, place them in a folder on your workstation alongside the Eye agent files.

terra_agent.py terra_launch.bat
3

Pull ODM + launch

Double-click terra_launch.bat. Terra will guide you through pulling the ODM image (one-time, ~2 GB), then open the platform automatically.

docker pull opendronemap/odm
python terra_agent.py
